- 角色自进化机制
- Argus MCP 集成与降级回退
- Tech Lead 自递归
- PowerShell 自动同步脚本 (scripts/sync.ps1)
- Completion Status Protocol (退出状态枚举)
- Scope Drift Detection (范围漂移检查)
